PLANIFICACIÓN Y DISEÑO DEL PROYECTO

En esta semana, nos centramos en la planificación y el diseño del proyecto. Definimos las características clave de nuestro sistema de riego automatizado y comenzamos a planificar los componentes electrónicos necesarios. En esta etapa, también decidimos cómo estructuraríamos el flujo de trabajo y qué herramientas usaríamos para la programación y las pruebas del sistema.

Se definieron los objetivos y las especificaciones del proyecto, asegurándonos de que cada componente del sistema tuviera una función clara y un propósito bien definido. También analizamos las posibles tecnologías que podríamos integrar, como los sensores de humedad y módulos de control remoto, para hacer el sistema eficiente y fácil de usar.

Selección de Herramientas y Componentes

Una de las decisiones más importantes de esta semana fue seleccionar las herramientas y componentes que formarían parte del sistema. Para la parte de hardware, decidimos usar un Arduino UNO como base de control. Elegimos un sensor capacitivo de humedad para medir la cantidad de agua en el suelo y un módulo relé para controlar la bomba de agua.

Además, discutimos las opciones de software y decidimos usar el IDE de Arduino para escribir y cargar el código al microcontrolador. También empezamos a desarrollar un esquema de conexión entre los diferentes componentes, como la bomba, el sensor de humedad y el Arduino, para que trabajaran de manera conjunta.
